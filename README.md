Project Description:
This project is a desktop-based Social Media Feed application developed in C++ using the SFML graphics library. It simulates an Instagram-like interface while implementing core Data Structures concepts
 ✨ Features:

- User Login
- Create Posts
- Like Posts
- Comment on Posts
- Share Posts
- Search Users
- Stories
- User Profiles
- Feed Navigation
- Attractive GUI using SFML
- Edit profile
 
 🧠 Data Structures Used:

- Queue

## ▶️ How to Run

### Requirements

- MSYS2 (UCRT64)
- C++ Compiler (g++)
- SFML Library
- VS Code (Optional)

### Steps

1. Open **MSYS2 UCRT64 Terminal**.

2. Navigate to the project folder:

```bash
cd /c/Users/IT-LINKS/Desktop/IRVINE/SocialMediaProject
```

3. Compile the project:

```bash
g++ Main.cpp User.cpp Post.cpp -o app.exe -lsfml-graphics -lsfml-window -lsfml-system
```

4. Run the application:

```bash
./app.exe
```

After running the above commands in the **MSYS2 UCRT64 terminal**, the Social Media Feed application will launch successfully.
